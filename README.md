# readfield

> 把读书会变成田野调查。

readfield 是一套可复用的读书会课程设计框架。它不教你"讨论一本书"，而是教你把一本复杂的文本当作一片待调查的田野，让参与者以田野调查者的身份进入、体验、沉淀。

## 核心理念

**不是"分析"，而是"体验"。** 参与者不是站在文本外面评论，而是被拽进文本里面，用自己的感官去感受那个世界。

这套框架的设计围绕几个关键机制：

- **跑团逻辑**：投入的前提不是假装，而是 stakes（选择有真实后果）
- **书记官角色**：领读人不是老师，是田野调查队的书记官，负责沉淀洞见、维护讨论纪律
- **集体田野笔记**：实时的、公开的、署名的笔记记录，让贡献被看见
- **入戏与出戏**：两者都是田野调查者的核心能力，不是对立关系

## 仓库结构

```
readfield/
├── docs/                  # 设计理念和方法论
├── templates/             # TeX 模板（课前材料、每日报告）
├── scripts/               # 自动化脚本（渲染、合并）
├── pipeline/              # 活动课竞赛管道（skill 设计竞赛）
│   └── data/              # 场景池、参赛 skills、运行结果
└── samples/               # 示例课程内容
    └── castle/            # 第一个完整案例：卡夫卡《城堡》
```
