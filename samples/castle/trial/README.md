# 试讲实例：行前准备

> 模拟圆桌试讲（4/5），2026-07-02
> 标题：准备！踏足于权力之野

## 基本信息

- **定位**：田野调查队的行前最后准备，不等同于正式课程中的某一天
- **时长**：45 分钟
- **目标**：让队员以具身性的方式理解田野调查的姿态，初次实践观察术

## 主题与视角

核心在"准备"二字。不推介田野调查，而是让大家在准备的过程中具身理解：
- 田野调查的姿态是什么样的？如何达到？
- 采取这种姿态会有什么后果？好处和潜在弊端？
- 带着怎样的问题意识进入田野，决定了后续如何开展工作、得出怎样的观察

总的问题意识：权力在城堡这片土地上采取哪些形式在运作，遵循什么规律。

## 课程流程（三部分，非常规顺序）

### 第一部分：前哨的信 — 角色入场

K 以队友身份（非队长）写信给调查队，分享自己先一步抵达城堡的所见所闻和感受。K 默认大家知道任务和使命，不训话不布置，而是在自然的叙述中让以下信息流露：
- 我们将去一个怎样的地方？
- 这个地方为什么值得去？被各种流言和解读笼罩的背后，有什么真实值得发掘？
- 我们期望在此次行动中获得什么？
- 将遭遇哪些可能的挑战？如何应对？

K 在信中会引用"别的调查队"得出的结论（寓言解读、精神分析解读等），并附上对应原文作为附件，支撑他的注释。

书记官按在信末引导，帮助大家完成从"不是队员"到"真正像队员"的转变，开始思考自己的使命和追求。当讨论形成共识，信的使命就达到了。

### 第二部分：对话触媒 — 掌握调查术

**顺序反常**：先发触媒，不发选段。

书记官向大家呈现一份田野调查教科书中标准策略与流程的选段，并说明：这是我们通行的工作方式，但需要根据我们调查的目标和地点做调整甚至重构。哪些坚持，哪些不再有效，需要以别的方式替代。

**重点引导**：将更多注意力放在观察术上。大家读文本本质上就是在体验、经历、观察，所以关键是：
- 关注什么，不关注什么？
- 记下什么，什么东西不重要？
- 捕捉在场而又不直观的东西的能力

**技巧要求**：不能直接说"你们不是真正在做田野调查，所以那些流程不用太关注"，要在不破角色的情况下引导注意力。

讨论完成后，大家再次发生转变：不仅是知道目标和使命的队员，还成为了真正理解调查技术、懂得观察术的队员。

### 第三部分：一手田野纪要 — 初次进入田野

大家既知道怎么做，又知道为什么做。现在真正上田野。

选段内容：K 刚到城堡时的田野纪实。大家带着已建立的问题意识和调查策略，初次体验城堡的文本，从方法论层面落到对问题实质的分析上。
